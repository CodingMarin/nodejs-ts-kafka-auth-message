## Usage

Steps to run this project:

###### Install dependencies

```bash
npm install
```

###### Run project

```bash
npm start
```

###### Run project in dev mode

```bash
npm start dev
```

###### Create migrations

```bash
npm run mig:gen -- src/migrations/"name-migration"
```

###### Run migrations

```bash
npm run mig:run
```

## Prettier

Format all files with Prettier:

```bash
npx prettier . --write
```
